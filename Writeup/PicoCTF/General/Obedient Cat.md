### Approach

**1. How to get the flag?**

First you need to download the file.

Seperti namanya, "Obedient **cat**", just run the ```cat``` command on the file like this:
```
cat file
```
And the flag will be shown here:

![flag](Obedient%20Cat-1.JPG)

The flag is:
```

```
